Create a modern, stylish, and high-end restaurant website focused on showcasing individual dishes instead of a traditional menu. The design should feel luxurious, artistic, and similar to Michelin-star or chef-driven concept restaurants.

Design style:
- Minimalist and elegant (black, white, neutral tones, gold accents)
- Large, high-quality images of dishes as the main focus
- Editorial / magazine-style layout
- Smooth animations and transitions (fade, parallax, scroll reveal)
- Strong visual storytelling

Main Sections:

1. Home
- Fullscreen hero with cinematic food video or image
- Restaurant name + short artistic slogan
- “Explore Dishes” and “Reserve” call-to-action buttons

2. Dishes (Main Feature)
- Grid or masonry gallery of individual dishes
- Each dish presented as a card with:
  - Name of the dish
  - Short creative description
  - High-quality image
- Click opens a detailed view with:
  - Ingredients
  - Chef notes / story behind the dish
  - Optional pairing suggestion (wine/drink)
- Smooth hover effects and transitions

3. Experience / Concept
- Explain the dining concept (seasonal, tasting experience, chef vision)
- Emphasis on creativity rather than fixed menu

4. Chef / About
- Chef profile with image
- Philosophy and inspiration

5. Reservation
- Elegant booking form (name, date, time, guests)
- Clean and simple UX

6. Contact
- Location, hours, Google Maps integration

UI/UX details:
- Focus on large imagery and whitespace
- Subtle animations (no clutter)
- Sticky transparent navbar
- Smooth scrolling
- Luxury feel similar to high-end restaurants

Tech requirements:
- Use React + Tailwind CSS (preferred)
- Component-based structure
- Reusable “DishCard” and “DishDetail” components
- Responsive design (mobile-first)

Bonus:
- Add filter by season or ingredient
- Add “Chef’s selection” highlight section
- Lazy loading for images
- SEO-friendly structure

Goal:
The website should feel like a curated culinary experience, where each dish is treated as a piece of art rather than part of a traditional menu.
